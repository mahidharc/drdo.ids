Home

This page informs the user about the Project Title, Team members and the Project Guide. This is the home screen of the application.


Start Testing

This page informs the user about the various datasets that the NBC can be tested against. The user will be provided a set of datasets to be selected to run the tests. After a particular dataset is selected, the NBC is tested against that dataset and the results will be provided. Various statistical visualizations can be seen to understand the results displayed.


Results

This page deals with the static results already obtained through constant experimentation. Results are displayed through various visualizations to help understand the user about the results of this experiment.
